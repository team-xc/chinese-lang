# 中文编程
基于 JavaScript 实现中文语言解释器

## 示例
```text
~ 这里不会被执行
~ 这里不会被执行

我有一个字符串「🐔」，等于“坤坤❤️”。
我有一个字符串「🏀」，等于“打篮球”。

弹窗：“我叫”「🐔」“，我会”「🏀」“。”。
弹窗：「🐔」。


我有一个整数「民族」，等于五十六。
控制台输出：民族“个民族”。
```

弹窗：
```text
我叫坤坤❤️，我会打篮球。
坤坤❤️
```

Console:
```text
五十六个民族
```
